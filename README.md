# ⚾ 숫자 야구 게임

터미널에서 즐기는 클래식 숫자 야구 게임입니다.

## 게임 방법

컴퓨터가 서로 다른 **세 자리 숫자**를 무작위로 정합니다.  
플레이어는 **10번** 안에 그 숫자를 맞춰야 합니다.

| 결과 | 의미 |
|------|------|
| 🟡 **스트라이크 (S)** | 숫자와 자리가 모두 일치 |
| ⚪ **볼 (B)** | 숫자는 있지만 자리가 다름 |
| ❌ **아웃 (OUT)** | 숫자가 하나도 없음 |

**3스트라이크** = 정답! 🎉

### 예시

```
정답: 4 2 7
입력: 4 1 2  →  1S 1B  (4는 자리까지 맞음, 2는 있지만 자리 다름)
입력: 9 2 7  →  2S     (2, 7은 자리까지 맞음)
입력: 4 2 7  →  3S 🎉
```

## 실행 방법

```bash
python game.py
```

Python 3.6 이상이면 별도 설치 없이 바로 실행됩니다.
